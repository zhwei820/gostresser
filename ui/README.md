# GoStresser

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run in docker

> 1. start gostresser backend https://github.com/hellofresh/gostresser;
> 2. docker-compose up;
> 3. visit http://localhost:8082/login in browser.
